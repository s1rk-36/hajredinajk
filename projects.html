WearMe

Developed a web application where users can enter an occasion they are attending and as a response they will see different ideas with Amazon links for each product. React and Next.js were used to develop the frontend. The backend was programmed using Flask where OpenAI API as well as Axesso, to make use of Amazon Product Advertising API, were used to generate the outfit ideas as well as the products.
